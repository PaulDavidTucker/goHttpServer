# goHttpServer
A simple Http server written in Go

## Features

- Handles HTTP requests
- Serves static files
- Simple routing

## Installation

```sh
git clone https://github.com/yourusername/goHttpServer.git
cd goHttpServer
go build
```

## Usage

```sh
./goHttpServer
```
